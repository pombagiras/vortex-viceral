# 🌑 Vortex Visceral 2026

> "Onde a carne encontra o código e o abismo manifesta a visão."

![Vortex Visceral Profile](https://raw.githubusercontent.com/pombagiras/melusine/refs/heads/main/Al%C3%AA%20(3)%20(3)%20(1).png)

**Vortex Visceral** é um oráculo digital de engenharia de prompts, projetado para manifestar estéticas gótico-abissais, rituais e biomecânicas. Desenvolvido por **Alexia Melusine**, o sistema utiliza uma matriz de pensamento sombrio para gerar comandos de alta complexidade destinados a IAs generativas como Midjourney (v6) e Leonardo.AI.

---

## 🔮 Funcionalidades

* **Invocação Estocástica:** Geração de prompts únicos combinando entidades arquetípicas, cenários desoladores e mestres do horror visual (Giger, Beksiński, Bacon).
* **Eternização no Limbo:** Integração direta com **Supabase** para armazenamento persistente das melhores visões.
* **Interface Abissal:** Layout mobile-first com feedback tátil (Haptic Feedback) e estética *dark-noir*.

## 🛠️ Tecnologias

* **Frontend:** HTML5, CSS3 (Visceral Aesthetic), JavaScript (Vanilla).
* **Backend:** [Supabase](https://supabase.com/) (Banco de Dados em Tempo Real).
* **Deploy:** GitHub Pages.

## 🕯️ Como Usar

1.  Acesse o [Vortex Visceral](https://pombagiras.github.io/vortex-viceral/).
2.  Toque em **"Invocar Visão"** para manifestar o prompt.
3.  Utilize **"Copiar"** para levar a semente ao seu gerador de imagem.
4.  Clique em **"Eternizar"** para gravar a visão permanentemente no banco de dados.
